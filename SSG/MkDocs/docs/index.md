# Azure 랜딩존 가이드

Azure 랜딩존(Landing Zone)은 클라우드 환경을 **안전하고 확장 가능하게** 구성하기 위한 기반 아키텍처입니다.  
이 문서는 엔터프라이즈 환경에서 Azure 랜딩존을 설계하고 구현하는 전 과정을 다룹니다.
