# 1. MariaDB 원격 접속 계정 생성 (사전 작업)

LB를 통해 들어오는 클라이언트가 MariaDB에 접속할 수 있도록 원격 접속 계정을 생성해야 합니다.

**node1 (maria01) 에서만 실행** → Galera가 나머지 노드로 자동 동기화

```sql
mariadb -uroot -p

-- LB 서브넷 또는 특정 IP에서 접속 허용
-- 예: 172.27.0.0/24 대역 허용
CREATE USER 'appuser'@'172.27.0.%' IDENTIFIED BY 'AppPass1234!';
GRANT ALL PRIVILEGES ON *.* TO 'appuser'@'172.27.0.%';
FLUSH PRIVILEGES;

-- 외부 클라이언트에서 직접 접속하는 경우 (공인 IP 대역 또는 % 사용)
CREATE USER 'appuser'@'%' IDENTIFIED BY 'AppPass1234!';
GRANT ALL PRIVILEGES ON *.* TO 'appuser'@'%';
FLUSH PRIVILEGES;
```

계정이 3개 노드에 모두 동기화되었는지 확인

```sql
-- node2 또는 node3 에서 확인
SELECT user, host FROM mysql.user WHERE user = 'appuser';
```
