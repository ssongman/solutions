# 2. 아키텍처 개요



- **구성**: 3 Node (권장 최소 구성)

- **특징**

  

  - Multi-Master (모든 노드 read/write 가능)
  - Sync Replication (데이터 즉시 동기화)
  - Quorum 기반 → 반드시 홀수 노드

  

# 3. 사전 준비



## 1) hostname 변경

```sh

hostnamectl set-hostname maria01
hostnamectl set-hostname maria02
hostnamectl set-hostname maria03

```



## 2) 서버 정보 확인

| **Node** | **IP**       | **Hostname** |
| -------- | ------------ | ------------ |
| node1    | 172.27.0.24  | maria01      |
| node2    | 172.27.0.53  | maria02      |
| node3    | 172.27.0.175 | maria03      |



## 3) /etc/hosts 설정 (모든 노드 동일)



```
vi /etc/hosts

172.27.0.24 maria01
172.27.0.53 maria02
172.27.0.175 maria03
```





## 4) 방화벽 오픈



```
firewall-cmd --permanent --add-port=3306/tcp
firewall-cmd --permanent --add-port=4567/tcp
firewall-cmd --permanent --add-port=4568/tcp
firewall-cmd --permanent --add-port=4444/tcp
firewall-cmd --reload
```



## 5) SELinux (PoC 기준 disable)



```
setenforce 0
vi /etc/selinux/config
SELINUX=disabled
```
